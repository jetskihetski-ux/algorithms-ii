Chinese Remainder Theorem

Given congruences x ≡ aᵢ (mod mᵢ) with pairwise coprime moduli, CRT guarantees a unique solution mod M = m₁·m₂·…·mₖ.

Construction: M = ∏mᵢ, Mᵢ = M/mᵢ, yᵢ = Mᵢ⁻¹ mod mᵢ
x = Σ aᵢ·Mᵢ·yᵢ (mod M)

The modular inverse yᵢ is found with the Extended Euclidean Algorithm.

Extended Euclidean Algorithm

Finds integers s,t such that gcd(a,b) = s·a + t·b. When gcd = 1, s is the modular inverse of a mod b.

def extgcd(a, b):
  if b == 0: return a, 1, 0
  g, s, t = extgcd(b, a%b)
  return g, t, s-(a//b)*t
# inv(a,m) = s when g=1
RSA-CRT Speedup

RSA decryption computes m = c^d mod n where n = p·q. CRT splits this into two smaller exponentiations:

mₚ = c^(d mod p-1) mod p
m_q = c^(d mod q-1) mod q
Combine via CRT → m mod n

Result: ~4× faster than direct computation via sub-problems of size ≈n/2.

Uniqueness Guarantee

When all moduli are pairwise coprime, CRT guarantees exactly one solution in [0, M). This is the foundation of residue number systems (RNS).

RSA-CRT Performance

Direct RSA decryption: one exponentiation mod n (1024+ bits). CRT splits it into two mod p and mod q (512 bits each). Since modular exponentiation is O(log²(n)) per multiply, we get roughly 4× speedup.

Applications

CRT appears in: RSA/smart-card crypto, error-correcting codes, fast integer arithmetic (NTT), secret sharing schemes, and calendar computations.
